# 计算机系统原理（2023年版）

## 第 1 章 计算机系统概述

### 1.1 计算机基本工作原理

**识记：**

中央处理器（CPU）：控制部件、运算部件和各类寄存器互连组成的电路  
算数逻辑单元（ALU）：进行算数逻辑运算的部件  
通用寄存器（GPR）：临时存放从主存取来的数据或运算的结果  
程序计数器（PC）：保存下一条指令的地址  
指令寄存器（IR）：保持从主存取来的指令  
控制器（CU）：自动逐条取出指令并进行译码的部件  
主存储器（MM）：存放指令和数据  
总线：链接不同部件进行信息传输的介质  
主存地址寄存器（MAR）：存放cpu送到地址线的主存地址  
主存数据寄存器（MDR）：存放发送或从数据线取来的信息
操作码字段：指出指令的操作类型  
地址码字段：指出指令所处理的操作数的地址  
机器指令：特定计算机规定的由0/1序列组成的指令
汇编指令：机器指令对应的符号表示  

**领会：**   
冯·诺依曼结构计算机的工作方式  

1. 采用“存储程序”的工作方式。基本思想是必须将事先编好的程序和原始数据送入主存后才能执行程序，一旦程序被启动，计算机不需要操作人员的干预就能自动完成逐条指令取出和执行的任务。
2. 计算机由运算器、控制器、存储器、输入设备和输出设备五大基本部件组成。
3. 存储器能存放数据，也能存放指令，在形式上没有区别，但计算机应能区分它们；控制器应能自动执行指令；运算器能进行基本算数和逻辑运算；操作人员可以通过输入/输出设备使用计算机。
4. 计算机内部以二进制形式表示指令和数据；每条指令由操作码和地址码两部分组成，操作码指出操作类型，地址码指出操作数的地址；由一串指令组成程序。


计算机硬件的基本组成  
cpu（包含运算器、控制器、各种寄存器）、主存、输入输出设备

程序和指令之间的关系  
程序由一串指令组成

指令的执行  
根据程序计数器取指令，指令译码，取数并执行、送结果、回到第一步开始循环
